HOW IT WORKS
================================================================================

CUSTOMER JOURNEY:

Day 1: Quote Sent
  → Customer receives email with portal link
  → Link contains unique token (UUID)
  → Token valid for entire job lifecycle

Day 5: Customer Opens Link
  → Portal loads with quote details
  → Shows status, amount, line items
  → Auto-refresh starts (every 30 seconds)

Day 10: Job Progresses
  → Status changes (quote → approved → scheduled)
  → Portal auto-updates without page reload
  → Messages appear in real-time
  → Files/photos visible

Day 20: Job Completed
  → Status changes to "completed"
  → Invoice appears in portal
  → Customer can see amount due
  → Payment button available

Day 25: Payment Received
  → Status changes to "paid"
  → Invoice shows "Paid" status
  → Token expiration set to 2 weeks from now
  → Customer can still view for reference

Day 39: Link Expires
  → Portal shows "Link Expired" message
  → Customer can't access anymore
  → Automatic cleanup complete
