### **Option 2: Manual SQL**

Run this in Supabase SQL Editor:

```sql

ALTER TABLE public.companies

-- Update existing companies
UPDATE public.companies 
SET timezone = 'America/Los_Angeles' 
WHERE timezone IS NULL;
```
